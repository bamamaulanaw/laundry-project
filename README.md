# Laundry Project

A simple CRUD (Create, Read, Update, Delete) web application for managing laundry orders using PHP Native and MySQL.

## Description

This application allows users to manage laundry transactions, including:
- Adding new laundry orders
- Viewing a list of orders
- Editing order details and status
- Deleting orders

The system automatically calculates prices based on service type and laundry weight.

## Requirements

- PHP 7.4 or higher
- MySQL
- Web server (Apache recommended)
- XAMPP or similar local server environment

## Installation

1. Start Apache and MySQL from XAMPP.
2. Create a database named `laundry_db`.
3. Create the table by running the provided SQL script.
4. Place the project folder inside:

```bash
C:\xampp\htdocs\laundry-project
```

5. Open the application in your browser:

```bash
http://localhost/laundry-project
```
